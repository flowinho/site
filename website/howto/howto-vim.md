# Befehle

|Befehl|Beschreibung|
|:-:|:----|
|`dd`|Löscht die ganze Zeile|
|`D`|Löscht vom Cursor bis zum Ende der Zeile|
|`d0`|Löscht vom Cursor bis zum Anfang der Zeile|
|`0`|Bewegt den Cursor zum Anfang der Zeile|
|`$`|Bewegt den Cursor zum Ende der Zeile|
